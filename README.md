### 创建一个Weex-APP

1，在安装Node.js的前提下，全局安装`ween-toolkit`

`sudo npm install weex-tookit -g`

`weex create awesome-app` 里面的选项尽可能选择YES

`cd awesome-app`

`sudo npm install`

`sudo npm start` 

`sudo weex platform add ios` 初始化iOS项目

`weex run ios`启动应用
